# University Management System (Backend)

This project is a backend system built using Java and Spring Boot to manage
university entities such as students, faculty, courses, and departments.

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- MySQL

## Features (In Progress)
- CRUD APIs for students, faculty, and courses
- Database integration using JPA
- Exception handling and validation

>  This project is currently under development.
